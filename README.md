# 端口管理工具

一个简单易用的端口管理GUI工具，可以查询端口占用情况并终止相关进程。

## 功能特性

- 🔍 **端口查询**: 查询指定端口的占用情况
- 🔄 **进程终止**: 安全终止占用端口的进程
- 📊 **批量显示**: 显示所有当前监听的端口
- 🎯 **PID操作**: 提取、复制和快速终止指定PID进程
- 🎨 **友好界面**: 简洁美观的图形用户界面
- ⚡ **一键启动**: 双击即可启动工具

## 使用方法

### 一键启动
双击 `启动工具.bat` 或 `启动端口管理工具.bat` 文件即可启动工具。

### 手动启动
```bash
python port_manager.py
```

## 依赖要求

- Python 3.6+
- psutil库（启动器会自动检查并安装）

## 界面说明

1. **端口号输入**: 输入要查询的端口号（1-65535）
2. **查询端口**: 查看端口占用情况和进程信息
3. **终止进程**: 终止占用指定端口的进程
4. **刷新**: 显示所有当前监听的端口
5. **PID快速操作**:
   - **提取PID**: 从查询结果中提取PID
   - **快速杀掉**: 直接根据PID终止进程
   - **复制PID**: 复制PID到剪贴板

## 安全提醒

- 终止进程前请确保了解该进程的作用
- 强制终止系统关键进程可能导致系统不稳定
- 建议在终止进程前保存相关工作的数据

## 技术支持

如有问题或建议，请检查：
1. Python环境是否正确安装
2. psutil依赖是否成功安装
3. 是否具有足够的系统权限
