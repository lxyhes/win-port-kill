# 🔌 端口管理工具 - 跨平台版本

一个功能强大的端口管理和进程监控工具，支持Windows和Mac双平台。

## 📁 项目结构

```
查杀软件工具/
├── Windows版本/                 # Windows专用版本
│   ├── port_manager.py         # 主程序
│   ├── 启动端口管理工具.bat     # ���动脚本
│   ├── 启动工具.bat            # 简易启动脚本
│   └── requirements.txt        # 依赖包
├── mac版本/                    # Mac专用版本
│   ├── port_manager_mac.py     # 主程序
│   ├── 启动端口管理工具.command # 启动脚本
│   └── requirements.txt        # 依赖包
└── README.md                   # 项目总览
```

## 🌟 功能特性

### 通用功能
- 🔍 **端口查询**: 查询指定端口的占用情况
- 🔄 **进程终止**: 安全终止占用端口的进程
- 📊 **批量显示**: 显示所有当前监听的端口
- 🎯 **PID操作**: 提取、复制和快速终止指定PID进程
- 📜 **端口历史**: 自动记录查询历史，支持快速重复查询
- ⚡ **一键启动**: 双击即可启动工具

### 平台特性

#### Windows版本
- 🪟 专为Windows优化
- 使用`netstat -ano`命令
- 现代化深色主题界面
- 批处理脚本启动

#### Mac版本
- 🍎 原生Mac风格设计
- 使用`lsof`和`netstat`命令
- SF Pro字体系统和Aqua主题
- Shell脚本启动

## 🚀 快速开始

### Windows用户
1. 进入 `Windows版本/` 目录
2. 双击 `启动端口管理工具.bat`
3. 或者运行 `启动工具.bat`

### Mac用户
1. 进入 `mac版本/` 目录
2. 双击 `启动端口管理工具.command`
3. 或者在终端运行：`python3 port_manager_mac.py`

## 📋 系统要求

### Windows
- Windows 7 或更高版本
- Python 3.6+
- psutil库

### Mac
- macOS 10.12 或更高版本
- Python 3.6+
- psutil库

## 🎮 使用方法

### 基本操作
1. **查询端口**: 输入端口号 → 点击查询
2. **管理进程**: 使用PID操作区域快速处理
3. **批量监控**: 刷新查看所有监听端口
4. **历史记录**: 快速重复查询常用端口

### 界面功能
- **端口操作区**: 查询、终止、刷新
- **PID快速操作**: 提取、杀掉、复制PID
- **历史记录**: 下拉选择和管理历史端口

## ⚠️ 安全提醒

- **了解进程**: 终止前了解进程作用
- **系统进程**: 避免终止关键系统进程
- **数据备份**: 操作前保存重要数据
- **权限要求**: 某些进程需要管理员权限

## 🛠️ 技术栈

- **语言**: Python 3.6+
- **GUI**: Tkinter
- **进程管理**: psutil
- **系统命令**: netstat/lsof
- **设计**: 现代化跨平台界面

## 📞 故障排除

### Windows问题
1. **Python环境**: 安装Python 3.6+
2. **依赖包**: 自动安装psutil
3. **权限**: 可能需要管理员权限

### Mac问题
1. **Python环境**: 使用`brew install python3`
2. **依赖包**: 运行`pip3 install psutil`
3. **权限**: 系统进程需要sudo权限
